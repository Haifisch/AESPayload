AESPayload Syringe
=================
AESPayload is a program used to help get to the AES engine to decrypt KBAG
keys. Easy (somewhat) program to use... Uses an iDevice for the engine so
have one ready to be used.

What devices can I use?
----------------------
*iPhone 3GS
*iPhone 3G
*iPhone 4 (GSM)
*iPhone 2G
*iPod 4G
*iPod 3G
*iPod 2G
*iPod 1G
